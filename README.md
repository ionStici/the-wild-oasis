# The Wild Oasis

Welcome to **The Wild Oasis** repository! This project is developed as part of a [course by Jonas Schmedtmann](https://www.udemy.com/course/the-ultimate-react-course/). It is a comprehensive web application designed for a hotel company to efficiently manage their operations, including customer check-ins, checkouts, and more.

This [project](<](https://the-wild-oasis.vercel.app)>) leverages a variety of modern web development technologies, including: React, Styled Components, Supabase, React Query, React Router DOM, React Hook Form, and more.
